Olist E-commerce Data Pipeline

From Raw Data to Business Impact
Turning messy e-commerce data into clean, ML-ready insights to predict delivery delays and improve customer satisfaction.

Project Overview

This project showcases my journey of building a real-world data pipeline on the Olist E-commerce dataset. Starting from raw CSVs, I learned to clean, transform, and engineer features that capture the complexity of online retail â€” from product weight and dimensions to shipping distance, cost, and customer location. The pipeline outputs both business-ready datasets with visual insights and ML-ready CSVs designed to predict delivery delays.

Along the way, I practiced core data handling with NumPy and Pandas, built business plots for customer and seller performance, and ensured data integrity through iterative debugging. Using an AI assistant as my co-developer, I also picked up frontend development basics to integrate these insights into a simple web interface.

At its core, this project tackles a real business need: understanding and reducing late deliveries, a critical factor for customer satisfaction and operational efficiency.
